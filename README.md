# Kalóz Dobókocka Beadandó

Egy egyszerű Android játék két játékos számára (Kalóz 1 és Kalóz 2), ahol a cél az, hogy az egyik játékos először gyűjtsön össze 100 aranyat dobókockával.

## Funkciók

- Véletlenszerű dobókocka dobás (1–6)
- Kör arany gyűjtése
- Kör elvesztése, ha 1-et dobsz
- Játékosváltás minden kör végén
- Pontok nyilvántartása
- Győzelem jelzése és új játék indítása
